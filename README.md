# cpp-mmaplib
C++11 header-only memory mapped file library, by [Yuji Hirose](https://github.com/yhirose).

## Installation

Run:
```bash
$ npm i cpp-mmaplib.cxx
```

And then include `mmaplib.h` as follows:
```c
#include "node_modules/cpp-mmaplib.cxx/mmaplib.h"
```
